Lead Lifecycle

What Happens to Every Lead

A step-by-step journey from capture to conversion

1
Lead Enters the System
Automatically from Facebook webhook, or manually added by admin.
📣 Facebook Ad Form ✏️ Manual Entry 📂 CSV Import
2
Lead Gets Assigned
Assignment rules auto-distribute to agents. Admin can also manually assign.
🔀 Round Robin 🎯 Source-Based 👆 Manual Pick
3
Agent Contacts the Lead
Agent opens lead detail page and uses quick-action buttons to reach the lead instantly.
📱 WhatsApp 📞 Call 📧 Email
4
Agent Updates Lead
Status updated, notes added, follow-up scheduled, tasks created — all in one page.
🏷️ Update Status 📝 Add Note 📅 Follow-up ✅ Create Task
5
Deal Created & Tracked
When lead is interested, agent creates a deal with value and moves it through pipeline stages.
💼 Lead → Qualified 📋 Proposal Sent 🤝 Negotiation
6
Admin Reviews Performance
Owner sees real-time dashboard with all metrics and can export reports anytime.
📊 Dashboard 📈 Reports 📥 Excel Export

Presentation Guide

Live Demo Script

Follow these steps to demo the CRM live to your owner

1

Login & Dashboard

Login as Org Owner. Show KPI cards, lead growth chart, agent performance table, and recent leads.

📊 /modules/dashboard/
2

Facebook Lead Sync

Go to Settings → Facebook Integration. Show connected pages and forms. Explain auto-webhook sync.

📣 /modules/facebook_integration/
3

Lead Assignment Rules

Settings → Assignment Rules. Show round-robin rule distributed equally to all agents automatically.

🔀 /modules/settings/assignment.php
4

Lead List & Filters

Open leads list. Show filters by status, source, agent, date. Show export to Excel button.

👥 /modules/leads/
5

Agent Works a Lead

Click any lead. Show hero banner, WhatsApp/Call/Email buttons, status update, add a note live, schedule follow-up.

🔍 /modules/leads/view.php
6

Deal Pipeline

Open Deals → Pipeline. Show stages and deal values. Demonstrate moving a deal from one stage to another.

💼 /modules/pipeline/
7

Agent Login View

Login as a sales agent. Show they only see their assigned leads. Add a note and schedule follow-up.

🧑‍💼 Agent account
